## 📝 Sobre o Projeto
Este sistema foi desenvolvido para automatizar o cálculo do salário líquido de funcionários. O projeto aplica conceitos de **Programação Orientada a Objetos**, separando a lógica de cálculo em classes específicas para garantir um código mais limpo e organizado.

## 🚀 Funcionalidades
* Cálculo automático com base em Salário Bruto, Descontos e Bónus.
* Interface Windows Forms intuitiva para entrada de dados.
* Implementação de classe `Funcionario` para gestão dos dados.

## 🛠️ Tecnologias
* C# (.NET Framework)
* Windows Forms
* Programação Orientada a Objetos (POO)
